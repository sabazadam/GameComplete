Game Collection Manager - Help Guide

Getting Started

Game Collection Manager is a tool for maintaining a catalog of your video games. You can add, edit, and delete games in your collection, as well as search and filter them based on various criteria.

Managing Your Collection

Adding a Game

To add a new game to your collection:

  1. Click the "Add Game" button in the toolbar, or select "Add Game..." from the Edit menu.
  2. Fill in the game details in the form. Fields marked with (*) are required.
  3. For list fields (genres, platforms, tags), enter values separated by commas.
  4. Click "Save" to add the game to your collection.

Editing a Game

To edit a game in your collection:

  1. Select the game in the table.
  2. Right-click and select "Edit Game", or select "Edit Selected Game..." from the Edit menu.
  3. Modify the game details in the form.
  4. Click "Save" to update the game in your collection.

Deleting a Game

To delete a game from your collection:

  1. Select the game in the table.
  2. Right-click and select "Delete Game", or select "Delete Selected Game..." from the Edit menu.
  3. Confirm the deletion when prompted.

Searching and Filtering

You can search for games in your collection by entering a search term in the search box and clicking the "Search" button. The search will look for matches in all fields of the games.

To clear the search and see all games again, clear the search box and click "Search".

Tip: You can search for games by any field, including title, developer, publisher, release year, tags, and more.

Importing and Exporting

Importing Games

To import games from a JSON file:

  1. Click the "Import" button in the toolbar, or select "Import..." from the File menu.
  2. Select a JSON file containing game data.
  3. The games will be imported into your collection.

Exporting Games

To export your collection to a JSON file:

  1. Click the "Export" button in the toolbar, or select "Export..." from the File menu.
  2. Choose a location and name for the JSON file.
  3. Your entire collection will be exported to the file.

JSON File Format

The application uses JSON files to store game data. Each game is represented as a JSON object with the following fields:

[
  {
    "title": "Game Title",
    "developer": "Developer Name",
    "publisher": "Publisher Name",
    "genres": ["Genre1", "Genre2"],
    "platforms": ["Platform1", "Platform2"],
    "translators": ["Translator1"],
    "steamId": "12345",
    "releaseYear": 2023,
    "playtime": 10.5,
    "format": "Digital",
    "language": "English",
    "rating": 9.5,
    "tags": ["Tag1", "Tag2"],
    "coverImagePath": "/path/to/cover.jpg"
  }
]

Keyboard Shortcuts

  • Ctrl+N: Add a new game
  • Delete: Delete the selected game
  • Ctrl+F: Focus the search box
  • Ctrl+O: Import games
  • Ctrl+S: Export games
  • Ctrl+H: Open help contents
  • Arrow keys: Navigate in the game list
  • Enter: Select the highlighted game
